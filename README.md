# Data Visualization

A comprehensive learning resource for mastering data visualization in Python using **Matplotlib** and **Seaborn**. This project contains hands-on Jupyter notebooks covering fundamental to advanced visualization techniques.

## Project Structure

```
Data-Visualization/
│
├── Matplotlib/
│   ├── 01_Matplotlib_Introduction.ipynb
│   ├── 02_Line_Plots.ipynb
│   ├── 03_Titles_Labels_and_Grid.ipynb
│   ├── 04_Markers_and_Styles.ipynb
│   ├── 05_Subplots_and_Figures.ipynb
│   ├── 06_Bar_Charts.ipynb
│   ├── 07_Scatter_Plots.ipynb
│   ├── 08_Pie_Charts.ipynb
│   ├── 09_Stack_Plots.ipynb
│   └── 10_Histograms.ipynb
│
├── Seaborn/
│   ├── 01_Seaborn_Introduction.ipynb
│   ├── 02_Seaborn_Datasets.ipynb
│   └── 03_Basic_Plot_Types.ipynb
│
├── README.md
└── requirements.txt
```

## Notebooks Overview

### Matplotlib Section (10 Notebooks)

| Notebook | Topics Covered | Cells |
|----------|---------------|-------|
| **01. Introduction** | What is Matplotlib, Basic setup, First plot | 3 |
| **02. Line Plots** | Creating line plots, Multiple lines, Customization | 22 |
| **03. Titles, Labels & Grid** | Adding titles, Axis labels, Grid lines, Legends | 23 |
| **04. Markers & Styles** | Line styles, Markers, Colors, Line width | 23 |
| **05. Subplots & Figures** | Creating subplots, Figure size, Layout management | 11 |
| **06. Bar Charts** | Vertical bars, Horizontal bars, Grouped bars, Stacked bars | 22 |
| **07. Scatter Plots** | Basic scatter, Colors, Sizes, Transparency | 18 |
| **08. Pie Charts** | Basic pie charts, Exploding slices, Custom colors | 23 |
| **09. Stack Plots** | Area plots, Stacked area charts, Cumulative data | 10 |
| **10. Histograms** | Distribution plots, Bins, Multiple histograms | 14 |


### Seaborn Section (3 Notebooks)

| Notebook | Topics Covered | Cells |
